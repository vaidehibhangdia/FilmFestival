# Film Festival Management System - Frontend

This is the React frontend for the Film Festival Management System.

## Features

- Manage Films
- Manage Attendees
- Manage Awards
- Manage Film Crew
- Manage Screenings
- Manage Tickets
- Manage Venues

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Backend

The backend is a Java application. To connect the frontend to the backend, the Java app needs to expose REST APIs.

## Technologies

- React
- React Router
- Bootstrap
- Axios

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
